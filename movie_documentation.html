Movie Ticket Booking Management System

Industry-level backend CRUD application using FastAPI, PostgreSQL, SQLAlchemy ORM and Pydantic Models for managing movie bookings, customer records and seat reservations digitally.

View Complete Repository

Problem Statement

Develop a backend Movie Ticket Booking Management System using FastAPI, PostgreSQL, SQLAlchemy ORM and Pydantic Models.

The system should perform complete CRUD operations for managing movie booking records digitally.

The application should store and manage:

• Booking ID
• Customer Name
• Movie Name
• Seat Number

Project Objective

Build scalable backend APIs capable of:

• Adding movie booking records
• Viewing booking details
• Updating booking information
• Deleting booking records

The system should validate incoming JSON data using Pydantic Models and store all records inside PostgreSQL database.

Functional Requirements

  • Add booking records using POST API
  • Fetch all booking records using GET API
  • Update movie and seat information using PUT API
  • Delete booking records using DELETE API

Technical Requirements

FastAPI PostgreSQL SQLAlchemy ORM Pydantic Uvicorn

Database Structure

Table Name:

MovieBooking

booking_id → Integer

customer_name → String

movie_name → String
